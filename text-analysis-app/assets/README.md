# Proyecto de Análisis de Texto

Este proyecto es una aplicación web que permite a los usuarios analizar el texto ingresado. Proporciona estadísticas útiles sobre el texto, incluyendo el número de caracteres, palabras, oraciones y un tiempo estimado de lectura.

## Estructura del Proyecto

- **src/index.html**: Contiene la estructura básica de la aplicación web.
- **src/styles.css**: Define los estilos CSS para la aplicación.
- **src/script.js**: Maneja la lógica principal de la aplicación y la interacción del usuario.
- **src/utils/textAnalyzer.js**: Exporta funciones para analizar el texto ingresado.
- **assets/README.md**: Información sobre los recursos utilizados en el proyecto.
- **README.md**: Documentación general del proyecto.

## Recursos Utilizados

- HTML5 y CSS3 para la estructura y el estilo de la aplicación.
- JavaScript para la lógica de la aplicación y la manipulación del DOM.
- Funciones personalizadas para el análisis de texto.

## Instrucciones

1. Clona el repositorio en tu máquina local.
2. Abre el archivo `src/index.html` en un navegador web.
3. Ingresa el texto en el área designada y observa las estadísticas generadas.
4. Utiliza los botones para resetear o copiar las estadísticas según sea necesario.

## Características

- Área de texto grande para ingresar texto.
- Cálculo automático de estadísticas de texto.
- Botones para resetear y copiar estadísticas.
- Interfaz de usuario intuitiva y fácil de usar.
